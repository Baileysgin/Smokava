# Logo Installation Instructions

To display your Smokava logo in the header:

1. Place your logo image file in this `public` folder
2. Name it one of the following (in order of preference):
   - `logo.svg` (best for quality)
   - `logo.png`
   - `logo-icon.svg`
   - `logo-icon.png`
   - `smokava.svg`
   - `smokava.png`

3. The logo should be:
   - Suitable for display at 40x40 pixels in the header
   - Transparent background (for SVG/PNG)
   - Contains your hookah logo design

The app will automatically detect and display your logo once the file is placed here.
